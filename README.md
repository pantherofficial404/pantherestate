# Panther Estate Web App

## Downloading
git clone https://github.com/pantherofficial404/pantherestate.git
cd pantherestate
pip install -r requirements.txt

## Functionality 
### Login
### Signup
### Admin Panel Access
### Sending Mail
### Adding Realtors
### Adding Listing Of Houses
### Most Valuable Person Of The Month
